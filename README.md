## 📌 Project Overview

This project presents a comprehensive **Digital Out-of-Home (DOOH) Advertising Performance Dashboard** built using **Google Looker Studio (formerly Data Studio)**. It provides campaign-level intelligence across multiple Indian cities, industries, and brands — enabling data-driven decisions for advertisers and media planners.

## 📊 Dashboard Features

- ✅ **Interactive filters** — Filter by City, Campaign, Industry, and Date Range
- ✅ **Revenue Over Time** — Line chart showing revenue trends across the year
- ✅ **Impressions Over Time** — Monthly trend with growth tracking
- ✅ **Industry Comparison** — Bar chart of revenue by industry
- ✅ **City Performance Map** — Revenue breakdown across 10 major Indian cities
- ✅ **Campaign ROI Leaderboard** — Top & Bottom 10 campaigns by ROI

## 🚀 How to Use

1. Click the **[Live Dashboard link](https://datastudio.google.com/reporting/c3181438-6a34-4cd3-8395-4bb6f93bf60d)** to open in Looker Studio
2. Use the **filter dropdowns** (City, Campaign, Industry, Date) to explore specific segments
3. Click **RESET** to clear all filters and return to the full view
4. For a static snapshot, refer to the **[PDF Report](https://github.com/prathmesh0work/DOOH-Dashboard/blob/dc4cfc7029ca336c3480524d6f94e2406927b31e/DOOH_advertising_Report%20(2).pdf)**
